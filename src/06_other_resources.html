The Halliday Centre Tagger (The Tagger)

The Halliday Centre tagger is a browser based tool to aid text analysis with automatic visualization and corpus. The Tagger provides customized feature which allows both pre-defined and self-defined tag for text analysis at different strata. For details, please refer to our step-by-step manual here.

Please register an account before your first use; you could find the registration form here (New User).

Internet connection is required for using the Tagger for the real-time corpus construction. The Tagger works best with Mozilla Firefox and Google Chrome. For technical support of the Tagger, please contact hcls@cityu.edu.hk.
